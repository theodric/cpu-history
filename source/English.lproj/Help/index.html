Mac OS X CPU Usage

CPU History is intended to mimic Apple's CPU Monitor application from the early days of OS X. The CPU usage of the system is broken into three parts: in use by user processes; in use by system processes; and idle CPU cycles.
Any program being executed under a user's ownership (such as Mail or Safari) is classified as a user process, and so its CPU usage is graphed as "user". Any program running as part of basic system functions (including the kernel) are classified as "system" CPU usage. Any CPU cycles that aren't in use by a user or system process are classified as "idle" usage.
